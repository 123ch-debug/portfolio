Projet POO - Gestion de Stock

Ce projet a été réalisé dans le cadre du module Programmation Orientée Objet. Il s'agit d'une application console permettant de gérer les clients, articles, commandes, fournisseurs, et factures. Le tout fonctionne avec des collections Java (HashMap, HashSet) et la sérialisation pour la sauvegarde des données.
