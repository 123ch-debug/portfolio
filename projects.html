Projet Java + MySQL - Application de Gestion de Stock

Ce projet a été développé en Java avec une base de données MySQL dans le cadre du module Projet de developpement SGBD Il s'agit d'une application graphique (Swing) permettant la gestion des clients, utilisateurs, articles en stock, commandes, ventes et paramètres de connexion. Trois types d'utilisateurs sont gérés (Admin, Gérant, Consultant) avec des droits d'accès différents. La base de données peut être recréée facilement grâce à un fichier SQL fourni.
